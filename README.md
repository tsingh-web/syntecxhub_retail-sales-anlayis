## Project Overview
This project analyzes retail sales data to identify sales trends, top products,
regional performance, and key business metrics.

## Objectives
- Perform sales analysis using a retail dataset
- Identify top products and seasonality
- Compute KPIs (Total Revenue, Average Order Value, Top Regions)
- Visualize sales trends and performance
- Export a one-page executive summary with charts

## KPIs
- Total Revenue
- Average Order Value (AOV)
- Revenue by Region
- Top Product Categories

## Tools Used
- Python
- Pandas
- Matplotlib
- VS Code
- Jupyter Notebook

## Visualizations
- Monthly Sales Trend
- Top Products by Revenue
- Revenue by Region

## Business Insights
- Sales show clear seasonal patterns
- Certain product categories dominate revenue
- Revenue is concentrated in specific regions

## Recommendations
- Increase inventory before peak months
- Focus marketing on high-performing products
- Expand operations in top regions

## How to Run
1. Clone the repository
2. Open `sales_analysis.ipynb` in VS Code or Jupyter
3. Run all cells
